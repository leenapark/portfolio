closetory

스타일링 관련 웹 사이트

GitHub: Closetory

옷을 잘 입는 사람도 못 입는 사람도 자신이 가지고 있는 아이템으로 스타일링 조언을 주고 받을 수 있는 웹 사이트 입니다. 몇 년 전 한 온라인 커뮤니티를 뜨겁게 달궜던 실시간 스타일링 글에서 아이디어를 얻었고 실제로 자신이 가지고 있는 아이템으로 스타일링을 주고 받음으로써 현실적으로 도움이 되고 싶다는 마음으로 개발했습니다.

처음 개발을 배우면서 진행한 프로젝트입니다. 개발의 기초부터 협업하는 과정에 관해 많이 배울 수 있었습니다.
만들어가는 과정 내내 서로 소통하였고 본인 파트가 아니어도 필요한 지식도 함께 공유하고 학습하였습니다.
회원 가입과 로그인 부분을 담당하였는데 이용자일 때는 단순하게 보였던 부분도 단순하지 않다는 걸 배울 수 있었습니다.

  • 주요 기능: 코디 추천 받기, 추천 하기, 스타일링 자랑하기, 옷 정보 공유하기
  • 언어: JAVA(JDK 1.8), HTML/CSS, JavaScript
  • 서버: Apache Tomcat 8.5
  • 프레임워크: Spring Framework 4.3.3, MyBatis 3.2.2
  • DB: Oracle 11g Enterprise
  • IDE: eclipse, sqldeveloper
  • API, 라이브러리: Jquery

아맞다! 여행

제주도 여행을 도와주는 웹 사이트

GitHub: 아맞다! 여행

여행객에게 필요한 부분을 세부적으로 나누어서 각 페이지를 나눴다. 크롤링, ml 을 사용할 수 있어서 무척 의미가 큰 프로젝트였다. 날씨 부분은 기상청 날씨누리을 실시간으로 크롤링해서 데이터를 보여주는 방식으로 진행하였습니다. 호스텔은 숙소 사이트를 크롤링하여 별점으로 세분화 하고 google JavaScript API 를 통해 지도를 그려 별점과 리뷰 데이터를 한번에 제공하는 방식을 사용하였습니다. 마지막으로 먹는 거에 진심인 한국인이 가장 중요하게 생각할 맛집 부분은 NAVER 지도에 제주도 맛집 인기순으로 정렬하여 리뷰와 별점을 수집하고 식당을 선택하면 해당 리뷰 리스트들을 이용자가 확인할 수 있다. 여기서 ml으로 예측한 부분은 수집한 데이터를 기반으로 새로운 데이터가 갱신되면 크롤링으로 받아온다고 가정하여 새로운 리뷰를 선택하면 실제로 이용한 사람이 호를 줬는지 불호를 줬는지 파악할 수 있도록 했다.

  • 언어: Python, HTML/CSS, JavaScript, jQuery
  • 프레임워크: Django
  • DB: SQLite
  • IDE: VS Code
  • API, 라이브러리: Ajax, fetch

O'CCU

SNS : Social Network Service

GitHub: O'CCU

O'CCU(이하 오꾸)는 SNS(Social Network Service) 사이트이다. K-POP이 세계적으로 인기를 끌고 있는 지금 아이돌 시장, 더 나아가 서브컬처(Subculture)는 더 이상 음지 취미가 아닙니다. 오꾸는 특정 관심사를 가진 사람들을 대상으로 한 SNS입니다. SNS을 프로젝트로 진행하면서 웹 어플리케이션 전체 틀을 JAVA로 구축하면서 멋모르고 배웠던 JAVA 언어를 더 깊게 이해하고 학습하는데 의미가 있었는데 ml/dl 의 강점을 가지고 있는 python으로 추천 모델을 만듦으로써 앞으로 뻗어나갈 인공 지능 분야의 무한한 가능성과 확장성을 몸소 체감할 수 있었습니다. 더 나아가면 모바일 서비스로도 만들 수 있는 주제라서 좋은 학습의 기회였습니다.

  • 주요 기능: 게시물 및 공유, 커뮤니티 형성, 이벤트 - 개인 창작자/구매자 연결
  • 언어: JAVA(JDK 1.8), Python, HTML/CSS, JavaScript
  • 서버: Apache Tomcat 8.5
  • 프레임워크: FastAPI, Spring Framework 4.3.3, MyBatis 3.2.2
  • DB: Oracle 11g Enterprise
  • IDE: eclipse, VS code
  • API, 라이브러리: Jquery, Pandas, numpy, joblib
